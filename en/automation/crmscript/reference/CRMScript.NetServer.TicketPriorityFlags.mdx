# Enum TicketPriorityFlags

Flags for the ticket priority

## Syntax

```crmscript

```

## Fields

| Name | Description |
|---|---|
| Unknown | No flags set |
| External | The ticket priority will be avilable in the Customer Centre |
| Default | This is the ticket priority that should default be set in the UI when creating a request |
| AlertSchedule | Escalate requests according to time frame |
