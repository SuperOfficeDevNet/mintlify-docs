You can change the width of the navigator by clicking above the navigator.
</Tip>

## The SuperOffice logo

At the upper left of the navigator is the SuperOffice logo. Click the logo to open the [SuperOffice Community website](https://community.superoffice.com/). Pressing **CTRL** while clicking the logo is a shortcut to your own company card. (The main company is a [preference](https://docs.superoffice.com/en/learn/getting-started/preferences.html) that you can change.)

The navigator provides quick access to essential SuperOffice CRM features. Each icon opens a specific screen.

|:---:|:---:|:---|

|  | **[Contacts](https://docs.superoffice.com/en/contact/learn/index.html)** | Access and manage **individual contact details**. |
|  | **[Diary](https://docs.superoffice.com/en/diary/learn/index.html)** | Plan and track **follow-ups**. |

|  | **[Selections](https://docs.superoffice.com/en/search-options/selection/learn/index.html)** | Create and manage **selections** for campaigns or workflows. |
|  | **[Inbox](https://docs.superoffice.com/en/email/inbox/learn/index.html)** | Read and reply to **incoming messages**. |

## Navigator options

When you click a navigator button, one or more of the following options are displayed:

| Icon | Function | Description |
|:---:|:---:|:---|
|  | **[History](https://docs.superoffice.com/en/learn/basics/history.html)** | Displays recently opened records, such as requests, contacts, or selections. |
|  | **[Favorites](https://docs.superoffice.com/en/learn/basics/fav.html)** | View the records you have set as favorites. |
|  | **Search** | Shows the results of the last search. The dynamic search field above the buttons initiates a search as you type. |

<Note>
**Note**

Clicking the **Dashboard** button only opens the dashboard.

If you hold the mouse pointer over a record in any the lists, a tooltip is displayed containing additional data about the record.
</Note>
