## setMarkdown(String)

Sets body text using Markdown syntax on dialog. If specified this will take precedence over setText().

```crmscript
Void setMarkdown(String value)
```

**Returns:** [CRMScript.Global.Void](CRMScript.Global.Void)

<Note>
&lt;b&gt;NOTE:&lt;/b&gt; This is not full Markdown but a \"lite\" version.
&lt;ul&gt;&lt;li&gt;# - Produces a &lt;h1&gt;&lt;/li&gt;&lt;li&gt;Newline in current paragraph - adds a &lt;br&gt;&lt;/li&gt;&lt;li&gt;Blank line - starts a new paragraph. Multiple blank lines in a row will create empty paragraphs.&lt;/li&gt;&lt;li&gt;Leading * - replaced with &bull;&lt;/li&gt;&lt;li&gt;Whitelisted tags are preserved (b, i, a, span, p).&lt;/li&gt;&lt;li&gt;Whitelisted attributes are preserved (class). To get a section styled differently, you may use a span element with a custom class name.&lt;/li&gt;&lt;/ul&gt;
</Note>

**Example:**

```crmscript
EventDataDialogDefinition def;
def.setMarkdown("[Google it](https://www.google.com)");
```
