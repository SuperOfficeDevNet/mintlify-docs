<h1>Anfrage</h1>

<p className="description">Anfrage ist die Hauptfunktion von SuperOffice Service. Hier verwalten Sie Anfragen und Rückmeldungen, die sich im System befinden. Sie können außerdem interne und externe Anfragen erstellen, eingehende Rückmeldungen beantworten und einen Überblick über die Status aller Anfragen im System erhalten. Die Anfragenverwaltung erfordert eine Service Essentials-Lizenz. Einige Funktionen erfordern einen Premium-Plan - siehe die Übersicht der Benutzerpläne für Details.</p>
