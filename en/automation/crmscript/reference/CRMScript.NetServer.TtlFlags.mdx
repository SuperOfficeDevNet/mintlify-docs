# Enum TtlFlags

Flag bits for extra information in TravelTransactionLog

## Syntax

```crmscript

```

## Fields

| Name | Description |
|---|---|
| None | No extra information. |
| Imported | The data change was caused by an Import operation. |
| MassOperation | The data change was done by a Mass Operation. |
