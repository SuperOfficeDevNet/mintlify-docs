# NSTicket

## Constructors

### NSTicket()

```crmscript
NSTicket
```

Initializes a new instance of the NSTicket class.

## Methods

## GetActivate()

```crmscript
DateTime GetActivate()
```

**Returns:** [CRMScript.Global.DateTime](CRMScript.Global.DateTime) - When the ticket should be activated, if it is postponed.

**Example:**

```crmscript
NSTicket thing;
DateTime activate = thing.GetActivate();
```

## GetAlertLevel()

```crmscript
Integer GetAlertLevel()
```

**Returns:** [CRMScript.Global.Integer](CRMScript.Global.Integer) - The alert level for the ticket. Matches the level value of the ticket_alert table.

**Example:**

```crmscript

Integer alertLevel = thing.GetAlertLevel();
```

