# SuperOffice som din løsning (macOS)

Dette er standardmåten for å koble e-postkontoen, dokumenter og kalender til SuperOffice på macOS.

<Note>
Disse instruksjonene gjelder kun for macOS. Bruker du en Windows-PC? Se [instruksjonene for Windows][9].
</Note>

## <a id="step-2"></a>E-post

<WhyConnectEmail />

Velg et av følgende alternativer:

* [SuperOffice Inbox][1] (standard).

* <WhySuperofficeForOutlook />

## <a id="step-3"></a>Dokument

Standard dokumentintegrasjon på macOS er **SuperOffice dokumentbibliotek** med [WebTools for Mac][2].

## <a id="step-4"></a>Kalender

Velg et av følgende alternativer:

* <SoInboxForInvitations />

* <WhySynchronizer />

* [WebTools for Mac][2] - for å eksportere din SuperOffice kalender til din iCalendar og kopiere dine SuperOffice kontakter til din Adressebok.

    <Note>
    Dette er **kun énveis** import, fra SuperOffice til din iCalendar, ikke en synkronisering!
    </Note>
