# Enum DocumentLockSemantics

Locking semantics requested/applied to a document

## Syntax

```crmscript
DocumentLockSemantics
```

## Fields

| Name | Description |
|---|---|
| None | Neither locking nor versioning is requested for the document |
| Locking | Locking, but not versioning, is requested for the document |
| Versioning | Versioning is requested for the document (locking is implied) |
