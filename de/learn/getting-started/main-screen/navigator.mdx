<Tip>
**Tip**

Sie können die Breite des Navigators ändern, indem Sie auf über dem Navigator klicken.
</Tip>
## SuperOffice-Logo

Oben links im Navigator befindet sich das SuperOffice-Logo. Klicken Sie auf das Logo, um die [SuperOffice Community-Website](https://community.superoffice.com/) zu öffnen. Wenn Sie beim Klicken auf das Logo die **STRG**\-Taste gedrückt halten, gelangen Sie direkt zur Firmenkarte Ihres Unternehmens. (Die Hauptfirma ist eine [Voreinstellung](https://docs.superoffice.com/de/learn/getting-started/preferences.html), die Sie ändern können.)

## Der Navigator im Standard-CRM

Der Navigator bietet schnellen Zugriff auf die wichtigsten Funktionen in SuperOffice CRM. Jedes Symbol öffnet eine bestimmte Ansicht.

| Symbol | Ansicht | Beschreibung |
| --- | --- | --- |
|  | **[Dashboard](https://docs.superoffice.com/de/dashboard/learn/index.html)** | Anzeigen von **Berichten und Diagrammen**, die CRM-Daten zusammenfassen. |
|  | **[Firma](https://docs.superoffice.com/de/company/learn/index.html)** | Verwaltung von **Firmendatensätzen** und zugehörigen Informationen. |
|  | **[Person](https://docs.superoffice.com/de/contact/learn/index.html)** | Zugriff und Verwaltung von **individuellen Personendaten**. |
|  | **[Kalender](https://docs.superoffice.com/de/diary/learn/index.html)** | Planung und Verwaltung von **Folgeaufgaben**. |
|  | **[Verkauf](https://docs.superoffice.com/de/sale/learn/index.html)** | Verfolgung und Verwaltung von **Verkaufschancen**. |
|  | **[Projekt](https://docs.superoffice.com/de/project/learn/index.html)** | Organisation und Überwachung von **Projektaktivitäten**. |
|  | **[Selektion](https://docs.superoffice.com/de/search-options/selection/learn/index.html)** | Erstellung und Verwaltung von **Selektionen** für Kampagnen oder Workflows. |
|  | **[Posteingang](https://docs.superoffice.com/de/email/inbox/learn/index.html)** | Lesen und Beantworten von **eingehenden Nachrichten**. |
|  | **[Chat](https://docs.superoffice.com/de/chat/learn/index.html)** | Beantwortung von **Chat-Anfragen** von Ihrer Webseite. |
|  | **[Marketing](https://docs.superoffice.com/de/marketing/learn/index.html)** | Erstellung und Versand von **E-Mail- und SMS-Kampagnen**. Formulare erstellen. Automatisierung. |
|  | **[Anfragen](https://docs.superoffice.com/de/request/learn/index.html)** | Verwaltung von **Service-Anfragen**. |
|  | **Werkzeuge** | Zugriff auf externe **Anwendungen und Webseiten**. |

## Navigatoroptionen

Beim Klicken auf eine Navigatorschaltfläche erscheinen folgende Optionen:

| Symbol | Funktion | Beschreibung |
| --- | --- | --- |
|  | **[Verlauf](https://docs.superoffice.com/de/learn/basics/history.html)** | Zeigt kürzlich geöffnete Datensätze wie Anfragen, Personen oder Selektionen. |
|  | **[Favoriten](https://docs.superoffice.com/de/learn/basics/fav.html)** | Zeigt Ihre favorisierten Datensätze. |
|  | **Suchen** | Zeigt Ergebnisse der letzten Suche. Die Suchleiste ist dynamisch. |
<Note>
**Note**

Durch Klicken auf **Dashboard** wird nur das Dashboard geöffnet.
</Note>
Wenn Sie den Mauszeiger über einen Datensatz bewegen, erscheint eine QuickInfo mit zusätzlichen Details.
