# Enum TaskDirection

Value for field 'direction' in table 'task'. Controls icons used in GUI

## Syntax

```crmscript
TaskDirection
```

## Fields

| Name | Description |
|---|---|
| Unknown | Unknown - used when initializing |
| Incoming | The task is sent in to SuperOffice CRM 5 |
| Outgoing | The task is sent out from SuperOffice CRM 5 |
| SaintAll | All' choice for Saint. This is NOT an acceptable value for a task definition, but is used by the Saint system for indexing all directions |
