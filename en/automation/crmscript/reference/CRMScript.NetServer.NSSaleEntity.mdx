# NSSaleEntity

The Sale Entity contains the sale amount, currency, and sale members. Sales are linked to contacts, persons, and/or projects.

## Constructors

### NSSaleEntity()

```crmscript
NSSaleEntity
```

Initializes a new instance of the NSSaleEntity class.
