# NSTicketPriorityEntity

Entity for a ticket priority. This entity describes the meta data for a ticket priority, and provides special operations on it.

## Constructors

### NSTicketPriorityEntity()

```crmscript
NSTicketPriorityEntity
```

Initializes a new instance of the NSTicketPriorityEntity class.

## Methods

## GetChangedOwner()

```crmscript
Integer GetChangedOwner()
```

**Returns:** [CRMScript.Global.Integer](CRMScript.Global.Integer) - What to do with the escalation chain when the request changes owner (manually). See &lt;xref href=\"CRMScript.NetServer.TicketPriorityEscalateEvent\" data-throw-if-not-resolved=\"false\"&gt;&lt;/xref&gt;.

**Example:**

```crmscript
NSTicketPriorityEntity thing;
Integer changedOwner = thing.GetChangedOwner();
```
