# NSUntrustedCredentials

Credentials for external authentication.

## Constructors

### NSUntrustedCredentials()

```crmscript
NSUntrustedCredentials
```

Initializes a new instance of the NSUntrustedCredentials class.

## Methods

## GetComment()

```crmscript
String GetComment()
```

**Returns:** [CRMScript.Global.String](CRMScript.Global.String) - Description of credentials. Max 255 characters.

**Example:**

```crmscript
NSUntrustedCredentials thing;
String comment = thing.GetComment();
```


## GetPublicValue()

```crmscript
String GetPublicValue()
```

**Returns:** [CRMScript.Global.String](CRMScript.Global.String) - Data stored unencrypted in the db.  Typically server and or username. Max 238 characters.

**Example:**

```crmscript
NSUntrustedCredentials thing;
String publicValue = thing.GetPublicValue();
```

## GetSecretValue()

```crmscript
String GetSecretValue()
```

**Returns:** [CRMScript.Global.String](CRMScript.Global.String) - Data stored encrypted in the db. Typically a password. Max 70 characters.

**Example:**

```crmscript
NSUntrustedCredentials thing;
String secretValue = thing.GetSecretValue();
```
