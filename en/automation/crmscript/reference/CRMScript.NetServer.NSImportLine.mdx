# NSImportLine

Used to import data into the system. Representing one entity that will be imported.

## Constructors

### NSImportLine()

```crmscript
NSImportLine
```

Initializes a new instance of the NSImportLine class.

## Methods

## GetExternalKey()

```crmscript
String GetExternalKey()
```

**Returns:** [CRMScript.Global.String](CRMScript.Global.String) - Optional external primary key for the row.

**Example:**

```crmscript
NSImportLine thing;
String externalKey = thing.GetExternalKey();
```

## GetOperation()

```crmscript
Integer GetOperation()
```

**Returns:** [CRMScript.Global.Integer](CRMScript.Global.Integer) - Which operation will be used? This is a read-only property. See &lt;xref href=\"CRMScript.NetServer.ImportAction\" data-throw-if-not-resolved=\"false\"&gt;&lt;/xref&gt;.

**Example:**

```crmscript
NSImportLine thing;
Integer operation = thing.GetOperation();
```

## GetSelected()

```crmscript
Bool GetSelected()
```

**Returns:** [CRMScript.Global.Bool](CRMScript.Global.Bool) - True if the entity shall be imported, false if the entity shall be ignored.

**Example:**

```crmscript
NSImportLine thing;
Bool selected = thing.GetSelected();
```

## GetType()

```crmscript
Integer GetType()
```

**Returns:** [CRMScript.Global.Integer](CRMScript.Global.Integer) - Which entity type will be created? This is a read-only property. See &lt;xref href=\"CRMScript.NetServer.ImportEntityType\" data-throw-if-not-resolved=\"false\"&gt;&lt;/xref&gt;.

**Example:**

```crmscript

```

## GetValues()

```crmscript
String[] GetValues()
```

**Example:**

