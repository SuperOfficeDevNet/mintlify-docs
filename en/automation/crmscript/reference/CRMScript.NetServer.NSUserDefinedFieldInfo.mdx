# NSUserDefinedFieldInfo

Can be used to mirror the layout of the user-defined fields in the user interface.

## Constructors

### NSUserDefinedFieldInfo()

```crmscript
NSUserDefinedFieldInfo
```

Initializes a new instance of the NSUserDefinedFieldInfo class.

## Methods

```crmscript

Integer columnId = thing.GetColumnId();

```
