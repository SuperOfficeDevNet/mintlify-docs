## Methods

## CreateDefaultSatellite()

Sets default values into a new NSSatellite.

```crmscript
NSSatellite CreateDefaultSatellite()
```

**Returns:** [CRMScript.NetServer.NSSatellite](CRMScript.NetServer.NSSatellite)

<Note>
NetServer calculates default values (for example, Country) on the entity, which is required when creating/storing a new instance
</Note>

**Example:**

```crmscript
NSReplicationAgent agent;
NSSatellite thing = agent.CreateDefaultSatellite();
thing = agent.SaveSatellite(thing);
```


**Returns:** [CRMScript.NetServer.NSSatellite](CRMScript.NetServer.NSSatellite) - The new or updated NSSatellite.
