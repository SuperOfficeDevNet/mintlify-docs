## GetAddress()

```crmscript
String GetAddress()
```

**Returns:** [CRMScript.Global.String](CRMScript.Global.String) - The address associated with this mailbox (used as FROM address when sending emails).

**Example:**

```crmscript
NSMailboxEntity thing;
String address = thing.GetAddress();
```

String GetPassword()
