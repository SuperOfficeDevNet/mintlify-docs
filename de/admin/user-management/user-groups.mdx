# Benutzergruppen verwalten

Um Kollegen zusammenzufassen und ihnen Zugriff auf CRM-Daten auf Grundlage dieser Gruppierung zu gewähren, müssen Sie in SuperOffice eine neue Benutzergruppe erstellen und diese dann Ihren Kollegen zuweisen.

## <a id="create"></a>Benutzergruppen erstellen

1. <GotoSm />

1. Wählen Sie <i class="ph ph-list-bullets" aria-hidden="true"></i> **Listen** im Navigator aus.

    ![Benutzergruppen erstellen -screenshot][img3]

1. Wählen Sie **Allgemein - Benutzergruppe** von der Liste aus und klicken Sie auf **Neu**.

1. Geben Sie Informationen in das Dialogfeld ein. Optional, wählen Sie eine oder mehrere [Anfragekategorie][4] für diese Gruppe.

    ![Benutzergruppen erstellen -screenshot][img4]

1. Klicken Sie auf **OK**. Die neue Benutzergruppe ist jetzt im Benutzer-Dialog verfügbar.

    ![Benutzer-Dialog -screenshot][img5]

## <a id="delete"></a>Benutzergruppe löschen

1. Gehen Sie wie oben beschrieben zur Liste **Allgemein - Benutzergruppe**.
1. Klicken Sie auf den Namen der betreffenden Benutzergruppe.
1. Klicken Sie unten in der Ansicht auf die Schaltfläche **Löschen**.
