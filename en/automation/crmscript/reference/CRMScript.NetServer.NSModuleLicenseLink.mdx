# NSModuleLicenseLink

Carrier object for ModuleLicenseLink.

## Constructors

### NSModuleLicenseLink()

```crmscript
NSModuleLicenseLink
```

Initializes a new instance of the NSModuleLicenseLink class.

## GetAssigned()

```crmscript
Bool GetAssigned()
```

**Returns:** [CRMScript.Global.Bool](CRMScript.Global.Bool) - Is the module license assigned to a given user.

**Example:**

```crmscript

```
